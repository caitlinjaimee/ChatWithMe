# ChatWithMe
ChatWithMe – Chatbot

 Overview
chatWithMe is a C# console-based chatbot designed to provide essential cybersecurity knowledge and its best practices. The application interacts in real-time, offering info on topics such as password security, phishing detection, safe browsing, and more.

 Features Of Console
	•  Interactive chatbot conversation
	•  (chatWithMe branding)
	• Typing animation f
	• Color-coded responses:
		○  Red → Threats (phishing, scams, malware)
		○  Green → Security practices
		○  Yellow → Tips and system messages
		○ Cyan → General information
	• Chat history tracking
	•  Keyword-based and exact-match response system
	•  Object-Oriented Programming (OOP) structure:
		○ Interface
		○ Base class
		○ Child class (inheritance & polymorphism)


How to Run
	1. Open the project in Visual Studio or VS Code
	2. Build the solution
	3. Run the program
	4. Interact with the chatbot via the console

 Example Commands
	• hello
	• help
	• password
	• phishing
	• best practices
	• show history
	• exit

 Project 
	• IChatbot → Interface defining chatbot behavior
	• ChatbotBase → Handles core responses
	• SmartCybersecurityChatbot → Adds advanced logic
	• Program → Main execution and user interaction
